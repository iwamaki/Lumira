# Lumira 文法ガイド

## 基本方針

- シンプルで学びやすい
- 日本語話者にも英語話者にも親しみやすい
- 柔軟性を持たせる（厳密すぎない）

## 語順

**基本語順: SVO（主語-動詞-目的語）**

```
Mi ama tu.
私 愛する あなた
= 私はあなたを愛しています
```

※語順は柔軟。強調したい要素を前に置くことも可能。

## 品詞

### 名詞

- 単数・複数の区別なし（文脈で判断）
- 冠詞なし

### 動詞

- 活用は最小限
- 時制は助詞で表現

| 時制 | 助詞 | 例 |
|-----|------|-----|
| 現在 | (なし) | Mi ama = 私は愛する |
| 過去 | -la | Mi amala = 私は愛した |
| 未来 | -va | Mi amava = 私は愛すだろう |

### 形容詞

- 名詞の前に置く
- 変化なし

```
Bela sola = 美しい太陽
```

## 否定表現（特殊ルール）

**Lumiraには直接的な否定語がない**

代わりに「まだ〜していない」「〜を待っている」などの
ポジティブな表現を使う。

| 通常の否定 | Lumiraでの表現 |
|-----------|---------------|
| 知らない | まだ学んでいる途中 (lerna-mi) |
| できない | これから習得する (vena-posa) |
| いない | 別の場所にいる (alta-loka) |

## 疑問文

文末に「-na」をつける

```
Tu ama mi-na?
あなた 愛する 私-？
= あなたは私を愛していますか？
```

## 接続詞

| Lumira | 意味 |
|--------|------|
| e | そして、と |
| o | または |
| ma | しかし（でも良い方向に） |
