# Lumira（ルミラ）- いい意味しかない架空言語

## プロジェクト概要

Lumiraは「いい意味しかない」をコアコンセプトとした架空言語です。
世の中のネガティブな言葉を、ポジティブな視点から再解釈することで、
言葉を通じて人々の心を明るくすることを目指しています。

## コアコンセプト

**すべての言葉は肯定的な意味を持つ**

- ネガティブな概念は存在するが、表現方法がポジティブ
- 言葉の響きは柔らかく心地よい
- 使う人の心が自然と前向きになる

## 言語名について

**Lumira（ルミラ）**
- lumi = 光（ラテン語 lumen より）
- ra = 流れ、広がり
- 「光が広がる」という意味を込めた
