# Lumira 音韻体系

## 設計思想

**柔らかく、心地よい響き**を最優先とする。
言葉を発するだけで気持ちが和らぐ音の組み合わせ。

## 母音 (5音)

| 文字 | 発音 | 例 |
|-----|------|-----|
| a | ア | alma (魂) |
| i | イ | mira (見る) |
| u | ウ | luna (月) |
| e | エ | bela (美しい) |
| o | オ | sola (太陽) |

## 子音 (15音)

### 優先使用（柔らかい音）

| 文字 | 発音 | 特徴 |
|-----|------|------|
| l | ラ行 | 流れるような音 |
| r | 軽いラ行 | 優しい響き |
| m | マ行 | 温かみ |
| n | ナ行 | 穏やか |
| s | サ行 | 静けさ |
| v | ヴァ行 | 風のよう |

### 通常使用

| 文字 | 発音 |
|-----|------|
| b | バ行 |
| d | ダ行 |
| f | ファ行 |
| h | ハ行 |
| k | カ行 |
| p | パ行 |
| t | タ行 |
| y | ヤ行 |
| z | ザ行 |

## 音節構造

**基本パターン: (C)V(C)**

- 子音 + 母音 が基本
- 連続子音は避ける
- 語末は母音で終わることが多い

### 推奨される組み合わせ

```
la, li, lu, le, lo
ma, mi, mu, me, mo
na, ni, nu, ne, no
ra, ri, ru, re, ro
sa, si, su, se, so
```

## アクセント

- 基本は後ろから2番目の音節
- 強弱アクセント（高低ではない）

```
luMIra (ルミーラ)
aMAla (アマーラ)
```

## 禁止される音の組み合わせ

以下は響きが硬すぎるため避ける：
- kk, tt, pp（重子音）
- kr, tr, pr（子音連続の一部）

## 音象徴（サウンドシンボリズム）

| 音 | 連想される意味 |
|----|--------------|
| l- | 光、流れ、愛 |
| m- | 包容、母性、温もり |
| s- | 静寂、星、精神 |
| a | 開放、始まり |
| i | 純粋、上昇 |
| o | 完全、円環 |
