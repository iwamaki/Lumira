# Lumira 言語コンセプト

## 核心理念

### なぜ「いい意味しかない言語」なのか

現代社会は言葉のネガティブさに溢れています。
ニュース、SNS、日常会話——私たちは無意識に
否定的な言葉を浴び続けています。

Lumiraは、言葉そのものを変えることで、
思考と感情のパターンを変えることを目指します。

## ネガティブ概念の再解釈ルール

### 基本原則

ネガティブな概念を「否定」するのではなく、
その中にある肯定的な側面を表現する。

### 変換例

| 従来の概念 | Lumiraでの解釈 | 理由 |
|-----------|---------------|------|
| 悲しみ | 心が深く感じている状態 | 感受性の豊かさを示す |
| 怒り | エネルギーの高まり | 変化を起こす力がある |
| 失敗 | 学びの種 | 成長の必須過程 |
| 死 | 大いなる旅立ち | 変容と継続 |
| 恐れ | 慎重さの声 | 自己保護の知恵 |
| 孤独 | 自分と向き合う時間 | 内省の機会 |
| 別れ | 新しい物語の始まり | 変化と成長 |

## 言語デザイン指針

### 音の選択

心地よい響きを優先する：
- 母音を多く使う（a, i, u, e, o）
- 流音（l, r）と鼻音（m, n）を多用
- 硬い破裂音（k, t, p）は柔らかい音と組み合わせる

### 語形成

- 短く、覚えやすい
- 意味と音の印象が一致する
- 複合語で新しい概念を表現可能
