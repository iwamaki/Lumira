# Lumira 例文集

## 日常会話

### 挨拶

| Lumira | 発音 | 意味 |
|--------|------|------|
| Sola! | ソラ！ | こんにちは！ |
| Luna! | ルナ！ | ありがとう！ |
| Bela vita! | ベラ ヴィタ！ | さようなら！（素敵な人生を！） |
| Pasa noi. | パサ ノイ | 私たちに平和を（別れ際） |

### 自己紹介

```
Sola! Mi [名前].
ソラ！ミ [名前]。
こんにちは！私は[名前]です。
```

### 感情表現

```
Mi senti profunda.
ミ センティ プロフンダ。
私は深く感じている。（= 悲しいけど、それは感受性の証）

Mi senti energa.
ミ センティ エネルガ。
私はエネルギーに満ちている。（= 怒りを感じるが、それは変化の力）

Mi flori.
ミ フローリ。
私は成長している。
```

## ポジティブ変換の例

### 従来: 「失敗した」
```
Mi planta sema.
ミ プランタ セマ。
私は種を植えた。（失敗は次への種まき）
```

### 従来: 「わからない」
```
Mi lerna-mi.
ミ レルナミ。
私はまだ学んでいる途中。
```

### 従来: 「彼は死んだ」
```
Il viaja la granda.
イル ヴィアハ ラ グランダ。
彼は大いなる旅に出た。
```

## 励ましのフレーズ

```
Tu bela kora.
トゥ ベラ コラ。
あなたは美しい心を持っている。

Noi flori sama.
ノイ フローリ サマ。
私たちは一緒に成長している。

Vita dona, vita ama.
ヴィタ ドナ、ヴィタ アマ。
人生は与え、人生は愛する。
```

## 詩的表現

### 朝の祈り
```
Sola nova,
vita nova,
kora flori.

新しい太陽、
新しい命、
心が花開く。
```

### 夜の感謝
```
Luna mira mi,
mi mira luna.
Pasa dormi.

月が私を見守り、
私は月を見上げる。
穏やかな眠りを。
```

## 練習問題

### 以下を Lumira に変換してみよう

1. 「おはよう、今日も頑張ろう」
2. 「あなたに会えて嬉しい」
3. 「困難があっても大丈夫」

### 回答例

1. `Sola! Noi flori nova.`（こんにちは！私たちは新たに成長する）
2. `Mi mira tu, mi senti bela.`（あなたを見て、美しく感じる）
3. `Sema flori.`（種は花開く）
